ДЛЯ СТУДЕНТОВ И ВЫПУСКНИКОВ
С ИНВАЛИДНОСТЬЮ 
Сервис представляет собой базу вакансий для людей с инвалидностью, 
предназначенную для анализа (просмотра) пользователями вакансий
по 7 территориям:
  • Кировская область
  • Оренбургская область
  • Республика Татарстан
  • Республика Удмуртия
  • Республика Чувашия
  • Республика Марий Эл
  • Республика Башкортостан
Пользователи могут не только проанализировать перечень профессий, востребованных на конкретный период, который отражен параметром
«дата обращения к вакансии», но и размер заработной платы и тип занятости. А также направление подготовки, которое необходимо
для освоения данной профессии.

Для составления базы данных востребованных вакансий для людей с инвалидностью были проанализированы несколько сайтов:
Trudvsem.ru, HH.ru, Superjob, Зарплата.ру, ГородРабот.ру. Основой для составления базы стали два основных сайта – Trudvsem.ru, HH.ru,
которые предоставляют пользователям возможность поиска работы по опции «для людей с инвалидностью», в отличии от сайтов Superjob,
Зарплата.ру, ГородРабот.ру, которые такой возможности не дают. 

В базу включены параметры востребованной профессии, заработной платы, типа занятости.
Также в базе представлено направление/я подготовки, необходимое для получения конкретной профессии.
Соотнесение профессии и направления подготовки осуществлялось на основе экспертного мнения.
Ряд профессий (специалист службы поддержки, тьютор, эксперт и др.), не позволяют определить конкретное направление подготовки и обозначены в базе данных строчкой «без предъявления требований к направлению подготовки высшего образования».    
Профессиональный стандарт также является одним из параметров базы данных и дает пользователям возможность познакомиться с трудовыми функциями, необходимыми для работы по выбранной профессии.
